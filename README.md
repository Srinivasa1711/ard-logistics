# ard-logistics

This repository contains a structured and simplified version of projects developed during my internship at ARD Logistics. The work focuses on building backend systems, dashboards, and workflow automation tools using Python, Flask, and SQL.

---

## Overview

The project includes two major systems:

1. KPI Dashboard System  
2. Training Log Management System  

These systems were designed to improve operational efficiency, automate manual processes, and provide better visibility into data for decision-making.

---

## 1. KPI Dashboard Development

During my internship, I developed internal KPI dashboards to replace an outdated tracking system.

### Key Highlights
- Built backend APIs using Flask to process operational data
- Integrated SQL-based data storage and retrieval
- Designed endpoints to serve real-time KPI data
- Supported frontend dashboards for visualization

### Impact
- Enabled real-time monitoring of key performance metrics  
- Improved decision-making speed and accuracy  
- Reduced reliance on manual reporting processes  

---

## 2. Training Log System

I designed and implemented a digital training log system to replace manual tracking workflows.

### Key Highlights
- Developed backend system using Python Flask and SQL
- Created structured workflows for training data tracking
- Implemented API-based communication between users and system
- Supported admin interfaces for managing training records

### Impact
- Centralized training data management  
- Improved tracking of user performance and progress  
- Reduced manual errors and increased efficiency
